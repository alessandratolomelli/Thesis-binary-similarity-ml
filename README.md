# BINARY SIMILARITY: A Comparative Analysis of Traditional Approaches and Machine Learning Methods

**BSc Thesis in Computer Engineering** **University of Bologna (Alma Mater Studiorum)** – A.A. 2024/2025

---

## Project Overview
This thesis explores the challenge of **Binary Similarity Analysis**, which aims to determine if two executable files originate from the same source code without access to the original source. This task is crucial for malware detection, vulnerability research, and software copyright protection.

### Key Areas of Focus:
* **Traditional Methods:** Analysis based on Control Flow Graphs (CFG), graph isomorphisms, and static heuristics.
* **Machine Learning Approaches:** Leveraging Deep Learning, specifically **Transformers** and **Graph Neural Networks (GNNs)**, for semantic feature extraction and embedding generation.
* **Comparative Study:** Evaluating the trade-offs between computational efficiency and accuracy across different datasets.
